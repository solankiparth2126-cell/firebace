# **App Name**: MoneyFlow

## Core Features:

- Transaction Input: Capture income and expense transactions with details like date, category, and amount.
- Ledger Management: Create and manage multiple ledgers for different accounts or purposes.
- Dashboard Overview: Display a summary of financial data, including balances, recent transactions, and spending trends.
- Reporting: Generate reports of spending habits with the help of a generative AI tool that can point out spending anomalies or suggest savings based on common sense personal finance principles.
- User Authentication: Secure user accounts with email and password-based authentication.
- Data Persistence: Store and retrieve financial data using Cloud Firestore.
- Dark/Light Mode: Allow users to switch between light and dark themes for optimal viewing experience.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) for a professional and trustworthy feel.
- Background color: Light gray (#F5F5F5) for light mode and dark gray (#303030) for dark mode.
- Accent color: Teal (#009688) to highlight important elements and CTAs.
- Body and headline font: 'Inter' (sans-serif) for a modern, neutral, and readable look.
- Use simple and clean icons from Material Design to represent transaction categories and actions.
- Employ a responsive layout with a sidebar navigation for easy access to different sections of the application. Utilize glass effect cards for a modern look.
- Incorporate subtle animations and transitions to enhance the user experience and provide visual feedback.
